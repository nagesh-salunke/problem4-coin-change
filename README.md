# Coin Change Minimum Coins

Given N coin denominations and a target amount X, find the minimum number of coins required to make X. Print -1 if it is impossible.

## Input
N X
coin1 coin2 ... coinN

## Constraints
1 ≤ N ≤ 10
1 ≤ X ≤ 1000
1 ≤ coin value ≤ 100

## Output
Minimum number of coins or -1

## Examples
Input:  
3 11  
1 2 5  
Output:  
3

Input:  
2 3  
2 4  
Output:  
-1
